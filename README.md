# 模拟Windows的一个简易记事本Qt项目
## 已完成部分
基础文件操作  
打开本地文本文件  
保存文件（支持首次保存时选择路径，已保存文件直接覆盖） 
关闭窗口时自动检测文件修改状态，提示是否保存（防止内容丢失） 

文本编辑功能  
基本文本输入、修改（基于 QTextEdit 组件）  
字体设置（通过字体对话框选择字体、大小等，并保存配置）  
字体缩放（支持 Ctrl+= 放大、Ctrl+- 缩小，以及 Ctrl + 鼠标滚轮 缩放，实时显示缩放百分比）  
光标位置显示（状态栏实时显示光标所在行号和列号） 

状态与交互提示  
窗口标题实时显示文件名及修改状态（修改后加 * 标记）  
状态栏显示当前文件名及修改状态  
菜单 / 快捷键支持（如 Ctrl+O 打开、Ctrl+S 保存、Ctrl+D 关闭等）
